| 按键、命令           | 功能            | 备注                                   |
| -------------------- | --------------- | -------------------------------------- |
| ctrl+e\y             | 向上\下滚动一行 |                                        |
| ctrl+b\f             | 向上\下滚动一屏 |                                        |
| ctrl+u\d             | 向上\下滚动半屏 |                                        |
| dd                   | 删除一行        |                                        |
| /str                 | 搜索str         | 搜索模式下N/n前后移动                  |
| 调到第一行和最后一行 | gg、G           | 命令模式输入行号并回车，跳转到任意一行 |
| ~/.vimrc             | vim配置文件     | 添加行号，更改配色，设置编码格式       |
